---------------------------
What is the NEEM Evaluator?
---------------------------

The NEEM Evaluator package is a collection of tools to evaluate and visualize NEEMs. It provides
different metrics for the experience and narrative parts of the NEEM.

----------------
What is a NEEM ?
----------------
NEEM stands for Narrative Enabled Episodic Memory. It is a knowledge representation for robots that
is based on the concept of episodic memory. NEEMs use the SOMA ontology to represent the knowledge
that is saved per episode. If you want to know more about NEEMs, please visit the
`NEEM Handbook <https://ease-crc.github.io/soma/files/current/NEEM-Handbook.pdf>`_.
